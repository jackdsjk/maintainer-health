# maintainer-health v0.1.0

Initial public release.

## Highlights

- Offline repository maintenance audit.
- Text output for humans.
- JSON output for automation.
- `--fail-under` threshold for CI gates.
- Checks for README, license, contributing guide, code of conduct, security policy, CI, tests, issue templates, pull request template, changelog, release metadata, and package metadata.

## Validation

- `python -m ruff check .`
- `python -m pytest`
- `python -m maintainer_health . --fail-under 90`

## Notes

The tool checks local repository files only. It does not prove adoption, project importance, or eligibility for external programs.
