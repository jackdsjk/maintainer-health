# maintainer-health v0.2.0

Larger feature release focused on making audits more useful across common project ecosystems.

## Highlights

- Detects Python, JavaScript, Rust, and Go repositories from local project files.
- Adds ecosystem-specific metadata checks.
- Adds Markdown report output with `--format markdown`.
- Shows detected ecosystems in text, JSON, and Markdown output.
- Updates package version metadata to `0.2.0`.

## Validation

- `python -m ruff check .`
- `python -m pytest`
- `python -m maintainer_health . --fail-under 90`
